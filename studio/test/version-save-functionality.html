Version Save Functionality Test

This test demonstrates how the save button becomes enabled when a version is selected.

Test the Save Functionality:

  1. Notice the "Save" button is initially disabled
  2. Click "Version History" to open the side panel
  3. Select a different version and click "Switch to Selected Version"
  4. Save button becomes enabled - you can now save the selected version
  5. Click "Save" to apply the version changes
  6. Save button becomes disabled again after saving
