        # Onboarding

Welcome to Merch at Scale project!

Any question you have, you can ask them on #merch-at-scale channel.

## you are a product manager

you probably need a new card layout that you want to see usable in m@s studio, please file a JIRA following that format <[....](/pages/createpage.action?spaceKey=adobedotcom&title=....&linkCreation=true&fromPageId=3375235501)>

Any question related to project should go in #merch-at-scale or #onboarding-<you>-mas.

## you are a developer

We have a strict **"no Direct Message"** policy as our team can't scale answering directly: if you feel your question is generic enough you can ask it on #merch-at-scale channel, if it's specific to your project or you have doubts ask it on #onboarding-<you>-mas.

You should have a "mas-<you>" branch on [https://github.com/adobecom/milo](https://github.com/adobecom/milo) project where work for your layouts has started. You can consume the [mas.js](https://www.adobe.com/libs/features/mas/docs/mas.js.html) library from that place: [https://mas-<you>--milo--adobecom.hlx.live/libs/features/mas/mas/dist/mas.js](https://mas-you--milo--adobecom.hlx.live/libs/features/mas/mas/dist/mas.js)

If you have any technical requirements for us, create a Jira ticket following that format [.....]
