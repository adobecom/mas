# Catalog Variant

## Overview

The Catalog variant is used for product catalog cards with action menus and detailed product information.

## Field Mappings

| Field | Slot | Tag | Size | Notes |
| --- | --- | --- | --- | --- |
| cardName | - | - | - | Sets name attribute |

## Slot Structure

```html
<slot name="icons"></slot>
<slot name="action-menu-content"></slot>
<slot name="heading-xs"></slot>
<slot name="heading-m"></slot>
<slot name="body-xxs"></slot>
<slot name="promo-text"></slot>
<slot name="callout-content"></slot>
<slot name="body-xs"></slot>
```
