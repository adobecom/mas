Version Switching Test

This test demonstrates how version switching updates the edit panel content.

How to Test:

  1. Click the "Version History" button in the right panel
  2. Select a different version from the list
  3. Click "Use Selected Version"
  4. Observe how the editor content changes to reflect the selected version
  5. Check the fragment info display to see the updated data
