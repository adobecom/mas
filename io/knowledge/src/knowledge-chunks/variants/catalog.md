# Catalog Card Variant

## Overview

The Catalog variant is the standard product card used on Adobe.com catalog and listing pages. It's one of the most commonly used variants for showcasing Adobe products with pricing information.

## When to Use Catalog

Use the Catalog variant when:
- Displaying products on catalog/listing pages
- Showing product pricing with a purchase CTA
- Creating product comparison layouts
- Building promotional product grids

## How to Create a Catalog Card in MAS Studio

1. Open MAS Studio and navigate to the **acom** surface
2. Click **"+ New Fragment"**
3. Select **Catalog** as the variant
4. Fill in the required fields (see below)
5. Save and preview your card

## Key Fields to Fill In

### General Info
- **Card Name**: Internal name for finding this card (e.g., "Photoshop-Catalog-US")
- **Style**: Choose Default (light) or Dark theme

### Visuals
- **Mnemonic Icon**: Click to select the product icon (Photoshop, Illustrator, etc.)
- **Badge**: Optional promotional text ("Best Value", "Most Popular", "New")
- **Size**: Default, Wide, or Super-wide layout

### Pricing
- **OST Button**: Click to search and select your product offer
- **Promo Code**: Enter if there's an active promotion
- **Show Strikethrough**: Enable to show original price crossed out during sales

### Content
- **Title**: Product name (e.g., "Adobe Photoshop")
- **Description**: Brief product description (keep under 150 characters)

### Footer
- **CTA Button**: Add your checkout button
  - Click "Add CTA"
  - Select button text (e.g., "Buy now", "Free trial")
  - Choose "Checkout" link type

## Best Practices

1. **Keep titles concise** - Product name is usually enough
2. **Use OST for pricing** - Never type prices manually
3. **Match icons to products** - Use the correct product mnemonic
4. **Test on mobile** - Preview how the card looks on smaller screens
5. **Use badges sparingly** - Only for genuine promotions

## Common Issues

### Price not showing
- Make sure you used OST to set the price, not manual text
- Verify the OSI code is valid for your locale

### Icon not appearing
- Check that you selected an icon from the mnemonic picker
- Some icons require specific product associations

### Card looks different than expected
- Check the Size setting (Default vs Wide)
- Verify Style setting matches your page (Light vs Dark)
