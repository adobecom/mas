# Ccd Suggested Variant

## Overview

The Ccd Suggested variant is used for suggested product cards for Creative Cloud Desktop.

## Field Mappings

| Field | Slot | Tag | Size | Notes |
| --- | --- | --- | --- | --- |
| backgroundImage | - | - | - | Sets background-image attribute |

## Slot Structure

```html
<slot name="icons"></slot>
<slot name="detail-s"></slot>
<slot name="heading-xs"></slot>
<slot name="body-xs"></slot>
<slot name="price"></slot>
<slot name="cta"></slot>
```
