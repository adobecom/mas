        # 1.1. Proposal Summary

Expand Merch @ Scale capabilities to XE platforms, fostering a cohesive user experience across different interfaces by utilizing current technologies. This expansion enables the activation of dynamic pricing and commerce functionalities throughout Adobe’s digital environment. For more details on Merch@Scale: [MWPPDM:Merch @Scale Phase 2](#)
