# Segment Variant

## Overview

The Segment variant is used for displaying merch card content.

## Field Mappings

| Field | Slot | Tag | Size | Notes |
| --- | --- | --- | --- | --- |
| cardName | - | - | - | Sets name attribute |

## Slot Structure

```html
<slot name="heading-xs"></slot>
<slot name="body-xxs"></slot>
<slot name="promo-text"></slot>
<slot name="callout-content"></slot>
<slot name="body-xs"></slot>
<slot name="badge"></slot>
```
