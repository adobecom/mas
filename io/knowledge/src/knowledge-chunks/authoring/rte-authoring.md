# RTE Fields in MAS Studio

## What are RTE Fields?

RTE (Rich Text Editor) fields let you format text and add special content like prices and links. When you see a text field with a toolbar above it, that's an RTE field.

## Where RTE Fields Appear

In the card editor, these fields have RTE toolbars:
- **Prices**: For adding pricing via OST
- **Description**: Main product description
- **Title**: Card heading
- **Promo Text**: Promotional messaging
- **What's Included**: Feature lists

## RTE Toolbar Buttons

### Text Formatting
- **Bold** (B): Make text bold
- **Italic** (I): Make text italic
- **Heading styles**: Change text size/style

### Content Insertion
- **Link button** (chain icon): Add hyperlinks
- **Shopping cart icon**: Open OST to insert prices
- **List button**: Create bulleted lists
- **Mnemonic button**: Insert product icons

---

## How to Add a Price in RTE

1. Click in the pricing field
2. Click the **shopping cart icon** in the toolbar
3. OST (Offer Selector Tool) opens
4. Search for your product
5. Select the offer
6. Click **Insert**

The price appears in the field automatically formatted.

**Important:** You cannot type prices as text. Always use the shopping cart button.

---

## How to Add a Checkout Button in RTE

1. Type your button text (e.g., "Buy now")
2. Select the text
3. Click the **link button** (or press Cmd+K)
4. In the link editor, choose **Checkout** as the link type
5. The link automatically connects to your product's checkout

---

## How to Format Text

### Make Text Bold
1. Select the text
2. Click **B** in the toolbar (or press Cmd+B)

### Make Text Italic
1. Select the text
2. Click **I** in the toolbar (or press Cmd+I)

### Create a Bulleted List
1. Click the **list button** in the toolbar
2. Type your first item
3. Press **Enter** for each new item

### Add a Link
1. Select text to turn into a link
2. Click the **link button** (or press Cmd+K)
3. Enter the URL
4. Click **Save**

---

## Tips for RTE Fields

1. **Keep formatting simple** - Too much bold loses impact
2. **Use OST for all prices** - Never type prices manually
3. **Short paragraphs** - Cards have limited space
4. **Test your links** - Preview to make sure they work

---

## Common Questions

### Why can't I type a price?
Prices must be added using OST (click the shopping cart icon). This ensures accurate, localized pricing.

### My formatting disappeared after save?
Some fields only support limited formatting. Title fields may strip bold/italic.

### How do I remove formatting?
Select the formatted text and click the formatting button again to toggle it off.
