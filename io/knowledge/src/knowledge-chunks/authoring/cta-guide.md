# CTA (Call-to-Action) Guide for Authors

This guide explains how to add checkout buttons and CTAs to your cards in MAS Studio.

---

## How to Add a Checkout Button in MAS Studio

Follow these steps to add a "Buy now" or any checkout button to your card:

### Step 1: Open the Footer Section
1. Open your card in the MAS Studio editor
2. Scroll down to the **Footer** section (or **CTA** section depending on variant)
3. You'll see the CTA configuration area

### Step 2: Add a CTA Link
1. Click **"Add CTA"** or the **"+"** button in the footer section
2. A new CTA field will appear

### Step 3: Configure the Button
1. **Button Text**: Select from the dropdown (e.g., "Buy now", "Free trial", "Learn more")
2. **Link Type**: Choose "Checkout" for purchase buttons
3. The checkout URL is automatically generated based on your OSI (Offer Selector ID)

### Step 4: Save and Preview
1. Click **Save** to save your changes
2. Click **Preview** to see how the button appears on the card

**Important:** The checkout button only works if you've set up pricing using OST (Offer Selector Tool). The button automatically links to the correct checkout flow for your selected product.

## Available CTA Options

| CTA Text | Best Used When... |
|----------|-------------------|
| Buy now | Direct purchase, user ready to buy |
| Free trial | Start a trial period |
| Start free trial | Emphasized trial start |
| Save now | Promotional pricing emphasis |
| Get started | Soft onboarding, low commitment |
| Choose a plan | Navigate to plan selection |
| Learn more | Educational, pre-decision stage |
| Upgrade | Existing customer upsell |
| Upgrade now | Urgent upsell messaging |
| Get offer | Claim a promotional deal |
| Select | Generic selection action |
| See more | Expand for additional info |
| Take the quiz | Interactive product finder |
| See all plans & pricing details | Navigate to full pricing page |

## CTA Selection Tips

### For Trial Offers
- Use **"Free trial"** or **"Start free trial"** for trial-focused cards
- These work best when the primary goal is getting users to try the product

### For Promotional Campaigns
- Use **"Save now"** or **"Get offer"** during sales events
- Creates urgency and emphasizes the deal

### For Existing Customers
- Use **"Upgrade"** or **"Upgrade now"** for upsell cards
- These are more direct for users who already have a subscription

### For Educational Content
- Use **"Learn more"** when users need more information before deciding
- Good for complex products or new users
