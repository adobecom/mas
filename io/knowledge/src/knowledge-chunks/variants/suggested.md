# Suggested Card Variant

## Overview

The Suggested variant is used for recommendation cards that highlight suggested products or upgrades. It's commonly used in upsell/cross-sell contexts where you want to show personalized offers.

## When to Use Suggested

Use the Suggested variant when:
- Recommending products based on user context
- Creating upgrade offers for existing customers
- Showing cross-sell opportunities
- Building personalized promotion cards

## How to Create a Suggested Card in MAS Studio

1. Open MAS Studio and navigate to the **ccd** surface
2. Click **"+ New Fragment"**
3. Select **Suggested** (or CCD-Suggested) as the variant
4. Fill in the fields (see below)
5. Save and preview

## Key Fields to Fill In

### General Info
- **Card Name**: Internal name (e.g., "AllApps-Upgrade-Suggestion")
- **Style**: Light or Dark theme

### Visuals
- **Mnemonic Icon**: Select the product icon
- **Badge**: Recommendation text ("Suggested for you", "Recommended")

### Content
- **Title**: Product name being suggested
- **Subtitle**: Tagline or secondary message
- **Description**: Why this is being suggested (value proposition)

### Pricing
- **OST Button**: Click to select your product offer
- **Show Strikethrough**: Enable to show original price crossed out
- **Promo Code**: If running a promotion

### Footer
- **Primary CTA**: Main action button (e.g., "Upgrade now")
- **Secondary CTA**: Optional second action (e.g., "Learn more")

## Best Practices

1. **Explain the value** - Why should the user care about this suggestion?
2. **Show savings clearly** - Use strikethrough pricing during promotions
3. **Personalize when possible** - "Based on your apps" messaging
4. **Limit suggestions** - Don't overwhelm users with too many options

## Common Use Cases

### Upgrade Suggestions
- Show higher-tier plans to current subscribers
- Highlight additional features they'd get
- Emphasize cost savings vs. buying separately

### Cross-Sell Recommendations
- Suggest complementary products
- "Add Lightroom to your Photography plan"
- Bundle opportunities

### Trial Prompts
- Suggest trying products they don't have
- "Try Illustrator free for 7 days"

## Common Issues

### Badge not visible
- Check that you entered badge text
- Some themes may hide badges

### Suggestion feels generic
- Add more context in the description
- Use personalized messaging when possible
