# M@S & Commerce Feature Requests

> Source: https://wiki.corp.adobe.com/pages/viewpage.action?pageId=3444176488
> Scraped: 2026-01-06T07:39:37.059Z

---


                           
        



Feature Requests

1. CTA Handling (disable, update label, etc.)
2. Localization/translation functionality within Studio
3. New commerce templates (see below)
4. Mobile responsiveness
5. Price comparison between two offers
6. Custom tags
7. Search fragments by tags, offerId
8. Team Support - quantity selector (see below)
9. Discount percentage support in Studio





## Feature Details

#1 - Support for an "undo" button experience. The user clicks on a button, the button's label updates to "undo". The button acts as a toggle.

#1 - Support for disabling CTA button(s) on click. Then enabling them through a function call.

#4 - Wide cards get truncated on mobile screens

#5 - Comparison between product in cart (e.g. PS) and recommendation (e.g. CC All Apps)

#6 - Ability for Studio user to specifcy custom tags like "Recommendation" or "Mobile", etc.

#7 - Ability to search for fragments based on tags and other fragment attributes.
