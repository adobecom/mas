# CCD Slice Variant

## Overview

The CCD (Creative Cloud Desktop) Slice variant is a compact, horizontal card used in Creative Cloud Desktop for upselling and product recommendations. It's designed to fit in narrow spaces while still conveying key information.

## When to Use Slice

Use the Slice variant when:
- Creating upsell offers in Creative Cloud Desktop
- Showing compact product recommendations
- Building inline promotions that don't interrupt workflow
- Displaying offers in sidebar-style layouts

## How to Create a Slice Card in MAS Studio

1. Open MAS Studio and navigate to the **ccd** surface
2. Click **"+ New Fragment"**
3. Select **Slice** as the variant
4. Fill in the fields (see below)
5. Save and preview

## Key Fields to Fill In

### General Info
- **Card Name**: Internal name (e.g., "Acrobat-Upsell-Slice")
- **Style**: Light or Dark theme

### Visuals
- **Mnemonic Icon**: Select the product icon
- **Badge**: Optional recommendation badge ("Recommended", "New")
- **Border Color**: Optional accent border

### Content
- **Title**: Product or offer name (keep very short - limited space)
- **Description**: One-line description only

### Pricing
- **OST Button**: Click to select your product offer
- **Promo Code**: If running a promotion

### Footer
- **CTA Button**: Single action button
  - Keep CTA text short (e.g., "Learn more", "Get it")

## Layout Notes

Slice cards are horizontal:
- Icon appears on the left
- Content flows left-to-right
- CTA button aligns to the right
- Very limited vertical space

## Best Practices

1. **Keep content extremely concise** - You have very limited space
2. **Use one CTA only** - No room for multiple buttons
3. **Short titles** - 3-4 words maximum
4. **One-line descriptions** - Or skip description entirely

## Common Issues

### Content gets cut off
- Your title or description is too long
- Shorten to fit the horizontal format

### Card looks cramped
- Remove optional fields like description
- Use shorter CTA text
