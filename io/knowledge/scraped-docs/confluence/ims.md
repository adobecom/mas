        You can create an IMS test account, e.g. [lukianet+de@adobetest.com](mailto:lukianet+de@adobetest.com):

- start with your ldap
- add + and any string
- email ending should be @adobetest.com

This account will work like normal adobe account and send emails to your ldap

For automation and reuse with other colleagues you might want to tag account as a test one, follow this wiki: [Tag test accounts - self service](/display/ims/Tag+test+accounts+-+self+service) 
Once tagged as test account, when logging in you won't be asked for a confirmation step via email/phone.

You can chose a country for your IMS account only while creating it.
